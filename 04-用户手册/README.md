# 数据直通车 · 用户手册（审计草案）

## 1. 文档说明

- 目标：为新用户、业务用户、治理用户和管理员提供统一上手材料。  
- 范围：覆盖系统定位、核心功能、设计理念、质量门禁、常见问题。  
- 读者：数据支持组、业务专家、一线业务人员、治理人员、系统管理员。  
- 当前阶段定位：审计前草案，仅沉淀内容，不直接对前端展示。

## 3. 使用约定

- 手册内容采用中文业务语境，技术名词只在必要处保留英文（如 LLM、SQL）。  
- 所有示例均以“业务场景”作为最小治理单元。

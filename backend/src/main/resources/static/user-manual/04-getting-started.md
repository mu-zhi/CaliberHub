# 04 主要功能上手指南

## 1. 文档说明

- 目标：提供从零到可发布的最短操作路径。  
- 范围：首页、口径治理、数据地图、个人中心、管理员能力。  
- 读者：全体角色。  
- 当前阶段定位：当前版本上手手册。  

---

## 2. 首页：统一入口

1. 在搜索框输入关键词，检索全局资产（当前以场景为主）。  
2. 通过“去导入口径文档”快速进入口径治理。  
3. 点击结果中的场景可直接进入编辑工作台。  

---

## 3. 口径治理：主流程

### 3.1 导入并生成草稿

1. 选择来源类型。  
2. 粘贴原始口径文档。  
3. 点击“导入并生成草稿”。  
4. 系统会自动完成预处理并创建草稿。

### 3.2 抽取质量判断

1. 查看综合分、结构完整性、SQL 保真率、关键字段完整度。  
2. 根据风险提示判断是否需要重点人工修订。  

### 3.3 抽取结果与原文对照

1. 左侧查看原文，右侧查看结构化结果。  
2. 对关键字段进行快速修订。  
3. 点击“同步到步骤 04”。  

### 3.4 场景编辑与发布

1. 选择业务领域并确认域概述。  
2. 维护输入参数、输出字段。  
3. 按方案维护 SQL（方案名、适用时段、来源表、SQL、备注）。  
4. 补充码值说明、风险提示、质量备注。  
5. 保存草稿。  
6. 填写验证时间和变更摘要后发布。  

### 3.5 发布检索

1. 按业务领域、状态、关键词查询。  
2. 列表按业务领域分组展示。  
3. 点击场景编号可回到编辑页继续处理。  

---

## 4. 数据地图：图谱浏览

1. 顶部：输入关键词并设置筛选条件。  
2. 左侧：从主题节点树进入业务上下文。  
3. 中间：查看资产关系图谱。  
4. 右侧：查看节点详情、字段规则、依赖影响、版本信息。  
5. 通过“浏览模式 / 路径模式 / 影响模式”切换观察角度。  

---

## 5. 个人中心：个人协作

1. 我的待办：查看当前待处理事项。  
2. 通知：查看系统更新与流程提醒。  
3. 收藏：管理高频场景入口。  
4. 最近浏览：快速回到最近访问的页面与场景。  
5. 系统管理（管理员）：维护大模型配置。

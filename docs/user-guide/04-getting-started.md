# 04 主要功能上手指南

## 1. 文档说明

- 目标：提供从零到可发布的最短操作路径。
- 范围：首页、知识生产台、数据地图、个人中心、管理员能力。
- 读者：全体角色。
- 当前阶段定位：当前版本上手手册。

---

## 2. 首页：统一入口

1. 在搜索框输入关键词，检索全局资产（当前以场景为主）。
2. 通过“进入知识生产台”快捷入口开始导入任务。
3. 点击结果中的场景可直接进入编辑工作台。

---

## 3. 知识生产台：主流程（当前上线 4 步）

### 3.1 导入并生成草稿

1. 选择导入内容类型和导入策略。
2. 上传文件或粘贴原始材料。
3. 点击“导入并生成草稿”。
4. 系统自动完成预处理并创建草稿。

### 3.2 抽取质量判断

1. 查看识别场景数、风险提示和低置信度项。
2. 在候选图谱中完成必要复核（接受 / 驳回 / 合并 / 拆分）。
3. 点击“确认质检，进入对照”。

### 3.3 抽取结果与原文对照

1. 左侧查看原文定位，右侧查看结构化预览。
2. 切换场景队列，逐个确认关键字段和取数方案。
3. 点击“确认对照，进入发布”。

### 3.4 场景编辑与发布

1. 选择业务领域并确认场景标题与描述。
2. 维护输入参数、输出字段与取数方案。
3. 补充码值说明、风险提示和质量备注。
4. 保存草稿。
5. 填写验证时间和变更摘要后发布。

### 3.5 发布后校验

1. 进入发布中心查看发布检查与快照状态。
2. 在数据地图确认关系是否符合预期。
3. 如涉及关键口径变更，补充质量备注并通知相关角色。

---

## 4. 数据地图：图谱浏览

1. 顶部：输入关键词并设置筛选条件。
2. 左侧：从业务领域进入业务上下文。
3. 中间：查看资产关系图谱。
4. 右侧：查看节点详情、覆盖检查、风险影响、版本信息。
5. 通过“业务视图 / 路径视图 / 覆盖视图”切换观察角度。

---

## 5. 个人中心：个人协作

1. 我的待办：查看当前待处理事项。
2. 通知：查看系统更新与流程提醒。
3. 收藏：管理高频场景入口。
4. 最近浏览：快速回到最近访问的页面与场景。
5. 系统管理（管理员）：维护大模型配置。

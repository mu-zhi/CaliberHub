## 一、测试目标

1. 可输出统一离线评测结果。

- 已覆盖：

2. 评测脚本当前默认跑 `RetrievalExperimentEvaluationService.main` 的 demo corpus；后续接真实回放集后，需要把样本输入外置成正式数据集。

# 口径文档质询维度（Question Taxonomy）

## 1. 语义歧义质询

关注点：

1. 码值是否有统一含义。  
2. 同名术语是否跨表同义。  
3. 时间口径是否存在自然语言歧义。

## 2. 多跳路径质询

关注点：

1. 主键与映射链是否完整。  
2. 跨域关联是否有明确路径。  
3. 路径选择是否可解释。

## 3. 复杂查询质询

关注点：

1. 嵌套、窗口、聚合是否存在口径冲突。  
2. 分段历史表路由是否稳定。  
3. 占位条件是否会导致错误。

## 4. 治理门禁质询

关注点：

1. 何时放行（allow，放行）。  
2. 何时复核（need_approval，需人工复核）。  
3. 何时阻断（deny，阻断）。

## 5. 审计追溯质询

关注点：

1. 是否有规则命中证据。  
2. 是否有追踪引用编号。  
3. 是否可回放到来源文档。

## 6. 评测稳定性质询

关注点：

1. 周级回放是否稳定。  
2. 误拦截率是否可控。  
3. 阈值漂移是否可观测。
